Welcome to this calculator I made. I based it on the iOS calculator (both in design and functionality). You can use it by clicking on the calculator or by typing. If you choose to type:

  • Hitting backspace deletes the last typed digit but does NOT clear the calculator. This is functionality limited to the keyboard.
  • The numbers correspond to numbers.
  • '*' and 'x' can be typed for multiplication. '/' can be used for division.
  • '+' and '-' can be used for addition and subtraction.
  • '%' can be used to get the percent value of the currently displayed number.
  • '_' can be used to make toggle the number between negative and positive values.
  • '=' and the return key can be used to compute the answer.
  • The escape key clears the calculator.
